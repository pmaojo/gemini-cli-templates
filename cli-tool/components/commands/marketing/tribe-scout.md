# Tribe Scout Command

Find the enemy (status quo).

## Parameters

- `topic`: The niche or topic to research.

## Usage

`gemini tribe:scout "topic"`

## Workflow

1. Use Google Search or Brave Search to find forums, Reddit threads, and community discussions.
2. Identify common frustrations and "status quo" beliefs.
3. Summarize the "Enemy" that the tribe is moving away from.
