# Tribe Launch Command

Draft the first 10 application process.

## Usage

`gemini tribe:launch`

## Workflow

1. Use the `marketing:culture` agent to define the initial onboarding experience.
2. Draft an application process for the first 10 members.
3. Define the success criteria for the "Minimum Viable Audience".
